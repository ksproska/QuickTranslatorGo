Start a service:
```cmd
make run-docker-compose
```

Template URL:
```
http://localhost:3000/youtube/{youtube_video_id}
```

Example YouTube video:
```
https://www.youtube.com/watch?v=se-KFMZLjXg&t=6s
```

Example URL for service:
```
http://localhost:3000/youtube/se-KFMZLjXg
```
